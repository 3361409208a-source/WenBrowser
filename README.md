# 🌊 MoyuBrowser (摸鱼浏览器)

![MoyuBrowser Logo](assets/logo.png)

> **为高效办公而生，让灵感在隐秘中自由流淌。**

MoyuBrowser 是一款专为办公室环境设计的“隐秘增强型”浏览器。它在提供流畅网页浏览体验的同时，集成了多项深度定制的隐秘功能，帮助开发者、设计师及广大职场人士在忙碌之余找到属于自己的呼吸空间。

---

## ✨ 核心特性

- **🛡️ 极致隐秘 (Stealth Shield)**
  - **全局老板键 (`Alt + B` / `Alt + Space`)**: 瞬间隐藏主窗口，从任务栏彻底消失。
  - **自适应淡化**: 当鼠标移出或失去焦点时，窗口自动调节至极低透明度，完美融入桌面壁纸。
  - **伪装身份**: 一键修改窗口标题为“财务审计报告”、“项目进度.docx”等，通过视觉伪装规避风险。

- **🎨 视觉美学**
  - **定制主题**: 内置 **VS Code 暗色**、**Office 简约白** 以及 **极简透明** 等多套主题。
  - **黑白模式**: 全局 Greyscale 滤镜，让网页内容回归纯粹，告别花哨，降低他人关注度。
  - **自定义字体**: 支持加载系统及外部字体，使网页排版更符合办公软件的既视感。

- **⚙️ 现代化交互**
  - **WebView2 驱动**: 采用 Microsoft Edge WebView2 核心，极速渲染，完美兼容所有现代网页。
  - **标签页强控**: 自动拦截外部弹窗，强制所有链接在内部标签页打开，保持任务栏整洁。
  - **动态热键**: 深度整合 Windows API，响应速度极快，手感丝滑。

---

## ⌨️ 快捷键指南

| 动作 | 快捷键 | 说明 |
| :--- | :--- | :--- |
| **老板模式** | `Alt + B` 或 `Alt + Space` | 瞬间隐藏/恢复主窗口 |
| **新建标签** | `Alt + T` | 在当前窗口打开新标签页 |
| **关闭标签** | `Alt + W` | 快速关闭当前活跃标签 |
| **亮度调节** | `Alt + ↑` / `Alt + ↓` | 动态增加/减少窗口透明度 |
| **快速最小化** | `Alt + Q` | 最小化至系统托盘 |

## 🚀 快速开始

### 系统要求
- Windows 10/11
- .NET 8.0 Runtime
- Microsoft Edge WebView2 Runtime

### 开发环境搭建
1. 克隆仓库: `git clone https://github.com/your-repo/MoyuBrowser.git`
2. 打开解决方案: 使用 Visual Studio 2022 或 VS Code。
3. 还原 NuGet 包并运行: `dotnet run`
